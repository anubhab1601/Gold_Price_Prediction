About This Project

Gold Price Prediction using Machine Learning

Project Overview

This web application predicts the GLD (Gold ETF) closing price using KNN Regression, trained on 7 years of daily market trading data (2011–2018).

Dataset

  • Source: FINAL_USO.csv
  • Records: {{ "{:,}".format(stats.total_records) }} daily rows
  • Date Range: {{ stats.date_start }} to {{ stats.date_end }}
  • Total Columns: 81 (11 used as features)
  • Target: Close (GLD ETF closing price in USD)
  • Missing Values: None
  • Duplicates: None

Best Model - KNN

  • Algorithm: K-Nearest Neighbours Regression
  • Optimal k: {{ stats.optimal_k }} (found via elbow curve, k=1–20)
  • Scaler: StandardScaler (fitted on train only)
  • Train / Test Split: 80% / 20%, random_state=0
  • R² Score: {{ stats.r2 }}
  • MAE: ${{ stats.mae }}
  • RMSE: ${{ stats.rmse }}

11 Features Used

  • SP_close - S&P 500 Index Close
  • DJ_close - Dow Jones Industrial Average
  • EU_Price - Euro / USD Exchange Rate
  • OF_Price - Oil Futures Price
  • SF_Price - Silver Futures Price
  • PLT_Price - Platinum Price
  • PLD_Price - Palladium Price
  • USDI_Price - US Dollar Index
  • GDX_Close - Gold Miners ETF (★ strongest)
  • USO_Close - Oil ETF Close
  • RHO_PRICE - Rhodium Price
